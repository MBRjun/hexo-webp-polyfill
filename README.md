# hexo-webp-polyfill
Hexo browser polyfill for the WebP image format.

## Installation
NPM
```
npm install hexo-webp-polyfill --save
```

Yarn
```
yarn add hexo-webp-polyfill
```

Enable the plugin in the main ``_config.yml`` file:
```yaml
webp_polyfill:
  enable: true
```
